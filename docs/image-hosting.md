# 图床方案选型

## Cloudflare R2（推荐）

| 项目 | 内容 |
|------|------|
| 免费额度 | 10GB 存储 + 无限出站流量 |
| 付费 | 存储 $0.015/GB/月，A类操作 $4.5/百万次 |
| 优点 | 出站流量免费，全球边缘缓存，SDK 成熟（S3 兼容） |
| 缺点 | 国内访问速度依赖 CF 网络，域名需备案 |
| 适合 | 中小站点，图片量不大 |

接入方式：S3 兼容 API（AWS SDK），后端生成预签名上传 URL 或直接通过凭证上传。

## 七牛云 Kodo

| 项目 | 内容 |
|------|------|
| 免费额度 | 10GB 存储 + 10GB/月 CDN 流量 |
| 付费 | 存储 ¥0.01/GB/天，CDN 流量 ¥0.28/GB |
| 优点 | 国内速度快，融合 CDN 覆盖好，有持久化处理 |
| 缺点 | 超出免费额后费用增长快，需绑定备案域名 |
| 适合 | 国内用户为主，流量较大的站点 |

接入方式：七牛 Java SDK（Java SDK for Kodo），服务端上传凭证。

## 阿里云 OSS

| 项目 | 内容 |
|------|------|
| 免费额度 | 无（新用户 3 月 20GB 试用） |
| 付费 | 存储 ¥0.12/GB/月，CDN 流量 ¥0.24/GB |
| 优点 | 国内节点多，功能全面，配合 ImageOptim 可实时处理图片 |
| 缺点 | 无长期免费额度，费用偏高 |
| 适合 | 已有阿里云资源，或需要图片处理管线的场景 |

## 又拍云

| 项目 | 内容 |
|------|------|
| 免费额度 | 无 |
| 付费 | 存储 ¥0.13/GB/月，CDN 流量 ¥0.29/GB |
| 优点 | 简单易用，文档齐全 |
| 缺点 | 无免费额度，单价较高 |
| 适合 | 已有又拍云账号的小站点 |

## 结论

**首推 Cloudflare R2**：10GB 免费存储 + 无限免费出站流量对资源论坛足够用。
**备选七牛云**：如果国内访问速度是瓶颈，七牛的 CDN 覆盖更好。

## 接入思路

后续实现时推荐方式：

1. 后端不直接存储文件，上传接口改为生成临时上传凭证
2. 前端直接上传到 OSS（避免经过后端中转）
3. 数据库中只保存文件 URL
4. 上传目录按场景划分：`avatars/`、`covers/`、`bg/`、`attachments/`
